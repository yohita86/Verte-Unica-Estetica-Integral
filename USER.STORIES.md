1. Registro de usuario
Como persona que desea atenderse en el establecimiento,
quiero poder registrarme en la plataforma con mis datos personales y credenciales,
para luego poder ingresar y gestionar mis turnos.

2. Inicio de sesión
Como usuario registrado,
quiero iniciar sesión con mi nombre de usuario y contraseña,
para acceder a mis turnos disponibles o agendar nuevos.

3. Agendamiento de turnos
Como usuario autenticado,
quiero agendar un turno eligiendo una fecha y hora disponible,
para asegurarme un espacio de atención en el establecimiento.

4. Cancelación anticipada
Como usuario autenticado,
quiero poder cancelar un turno hasta el día anterior,
para liberar ese horario si no voy a asistir.

5. Consulta de mis turnos
Como usuario autenticado,
quiero ver el listado de todos mis turnos (activos y cancelados),
para llevar el control de mis próximas citas.
